# ZsuTools
Полегшення життя діловода (ЄЖООС, стройова записка, БЧ, рапорти на премію та ДВ)

- Генерація рапорта на ДВ з табеля ЄЖООС
- Генерація рапорта на премію з табеля ЄЖООС (врахувати СЗЧ)
- Заповнення табеля ЄЖООС зі СЗ/БЧ
- Створення таймлайна бійця зі СЗ/БЧ (де і коли був, зміни звання)
- Карта бійця з ЄЖООС (ПІБ, вік, інша інфа з ООС, посада повна у відмінку)
- consider переміщення по посаді, коректно заповнити ООС, оновити Табель, ШПО
- Різні корисні кастомні функції для Excel (UDF)
    - IsGreen(), IsRed(), IsBlue(), IsBold(), IsStrikeout(), etc
    - consider коректно змінити посаду (взвод -> взводу, рота -> роти, командир взводу 1 взводу -> командир 1 взводу ...)
